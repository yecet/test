# CV Klasörü

Bu klasöre Oğuzhan Kapukaya'nın CV dosyasını ekleyin.

## Dosya Adı

```
oguzhan-kapukaya-cv.pdf
```

## Nasıl Güncellersiniz?

1. Yeni CV'nizi `oguzhan-kapukaya-cv.pdf` adıyla hazırlayın.
2. Bu dosyayı `public/cv/` klasörüne koyun (eskisinin üzerine yazın).
3. GitHub'a `git add`, `git commit`, `git push` ile gönderin.
4. Vercel otomatik olarak deploy eder.

## Mevcut Durum

Gerçek PDF henüz eklenmemiştir. Site şu an `/cv/oguzhan-kapukaya-cv.pdf` bağlantısını göstermektedir.
