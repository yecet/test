## Nasıl Eklenir?

1. PDF dosyasını bu klasöre koyun (yukarıdaki isimlendirmeye uygun).
2. `src/data/materials.ts` dosyasında ilgili `fileUrl` alanını kontrol edin.
3. GitHub'a push edin → Vercel otomatik deploy eder.
