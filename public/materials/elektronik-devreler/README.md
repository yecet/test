# Elektronik Devreler – Materyal Klasörü

Bu klasöre **EEF 301 Elektronik Devreler** dersine ait PDF materyalleri ekleyin.

## Beklenen Dosyalar

| Dosya Adı | Tür | Açıklama |
|-----------|-----|----------|
| `hafta1-slayt.pdf` | Slayt | Hafta 1 – Yarı İletken Fiziği |
| `hafta2-slayt.pdf` | Slayt | Hafta 2 – Diyot Modelleri |
| `odev1.pdf` | Ödev | Ödev 1 – Diyot Devreleri |
| `lab1.pdf` | Lab | Lab 1 – Diyot Karakteristikleri |

## Nasıl Eklenir?

1. PDF dosyasını bu klasöre koyun.
2. `src/data/materials.ts` dosyasındaki `fileUrl` alanını güncelleyin.
3. GitHub'a push edin.
